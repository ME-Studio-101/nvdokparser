## 🚀 Быстрый старт

```python
# 1. Импортируем необходимое
from sqlalchemy import Column, String, Integer
from engine.Data.database import init_db, get_db, CRUDBase
from engine.Data.database.models import BaseModel

# 2. Создаем модель
class User(BaseModel):
    __tablename__ = "users"
    username = Column(String(50), nullable=False)
    age = Column(Integer, nullable=True)

# 3. Инициализируем БД (один раз)
init_db()

# 4. Работаем с данными
db = get_db()
try:
    user_crud = CRUDBase(User, db)
    
    # Создаем пользователя
    user = user_crud.create({"username": "Иван", "age": 25})
    
    # Читаем пользователя
    found_user = user_crud.get(user.id)
    
    # Обновляем пользователя
    user_crud.update(user.id, {"age": 26})
    
    # Удаляем пользователя
    user_crud.delete(user.id)
finally:
    db.close()
```

## ❓ Частые вопросы

**Q: Когда использовать get_db() vs get_session()?**
A: `get_session()` рекомендуется - автоматически закрывает сессию. `get_db()` нужен, когда требуется прямой контроль.

**Q: Что такое commit()?**
A: `commit()` сохраняет изменения в БД. До commit() все изменения только в памяти.

**Q: Как создать свою модель?**
A: Наследуйтесь от `BaseModel`, определите `__tablename__` и добавьте поля через `Column()`.

**Q: Можно ли расширить CRUD?**
A: Да! Создайте класс, наследующийся от `CRUDBase[YourModel]` и добавьте свои методы.
